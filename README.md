# Conway's Game of Life

[See it in action](http://matthewboston.com/conway)

## Dependencies

- [boot](http://boot-clj.com/)

## Development

Start the `dev` boot task with:

`$ boot dev`

visit http://127.0.0.1:3000

## Testing

Run the tests once with:

`$ boot test-once`

Start the `test-auto` boot task with:

`$ boot test-auto`
